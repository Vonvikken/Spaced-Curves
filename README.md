# Spaced curves
A JavaFX application that shows a set of curves whose color bending, number and spacing can be set by parametrs.

## Requirements
* Java 13 (but can be run since Java 11)
* Maven

## Compile
`mvn compile`

## Run
`mvn javafx:run`
